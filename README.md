# Book Review Application

## Description
This is a web application that allows users to browse books, view details, and submit reviews. It features both general user and authenticated user routes, allowing for a range of functionalities.

## Features
- Browse a collection of books.
- View book details by ISBN, author, or title.
- User registration and authentication.
- Submit, edit, and delete book reviews.

## Technologies
- **Node.js**: Server-side JavaScript runtime.
- **Express**: Web framework for Node.js.
- **Axios**: Promise-based HTTP client for the browser and Node.js.
- **JSON Web Token (JWT)**: For secure user authentication.
- **Express Session**: Middleware for managing user sessions.
